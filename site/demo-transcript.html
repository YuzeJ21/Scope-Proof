ScopeProof 60-second demo

This is a deliberately constructed demo case. ScopeProof uses deterministic evidence rules and human review; it does not guarantee correctness or replace QA.

  1. 0:00–0:05 — The question. Does this pull request prove the requirement? ScopeProof maps acceptance criteria to auditable evidence.
  2. 0:05–0:15 — Confirm intent. A source owner confirms normalized criteria before analysis. ScopeProof never invents the requirement.
  3. 0:15–0:35 — Inspect evidence. Deterministic rules link implementation and test candidates to immutable file lines, or state what evidence is missing.
  4. 0:35–0:50 — Decide. A human inspects Missing, Partial, and Needs Review findings. Static evidence is never presented as runtime verification.
  5. 0:50–1:00 — Record the outcome. The result may be a useful gap, only known information, or friction. ScopeProof supports the review; it does not replace QA or guarantee correctness.
