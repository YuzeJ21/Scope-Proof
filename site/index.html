A bounded review

PR → Criteria → Evidence → Decisions → Outcome

01

PR

Load a public GitHub pull request without executing its code.

02

Criteria

The source owner confirms one atomic requirement per line before analysis.

03

Evidence

Deterministic rules cite immutable candidates or state what evidence is missing.

04

Decisions

A human accepts, rejects, verifies, or records an exception with a reason.

05

Outcome

Record one honest result without turning a trial into a market claim.

Help test the product question

Bring one genuine public PR

ScopeProof needs product managers, QA practitioners, and engineers who own or are directly authorized to confirm public acceptance criteria. A ten-minute trial can end honestly in one of three ways: it found a useful gap, showed only known information, or created friction.

Likes, views, stars, impressions, and downloads are not product validation.
