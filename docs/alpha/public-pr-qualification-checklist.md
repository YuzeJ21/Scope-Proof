# Public PR qualification checklist

A case counts as genuine public-alpha evidence only when every item is checked:

- [ ] The pull request is reachable without private-repository access.
- [ ] I am the source owner for the requirements, or I am directly authorized by that owner to confirm them.
- [ ] The requirements have a public HTTPS source that another reviewer can inspect.
- [ ] The criteria are confirmed before ScopeProof analysis begins.
- [ ] **No confidential information** will be pasted, stored, or exported.
- [ ] No tokens, secrets, customer data, private code, or private links are included.
- [ ] I understand that ScopeProof does not run repository code and does not replace QA.

If any item is unchecked, do not label the run as a confirmed public-alpha case. Use a clearly labeled technical smoke or decline the review.
